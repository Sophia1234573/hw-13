    В классе InteractionWithJsonPlaceholder содержаться методы для реализации указанного
в задании функционала при взаимодействовии с API https://jsonplaceholder.typicode.com.
    Для написания программы использовались стандартные возможности Java (HttpClient).

    Задание 1.
    
    Метод createNewUser(User user) добавляет новый обьект в в https://jsonplaceholder.typicode.com/users.
Объект, который нужно добавить, создается при помощи createDefaultUser() и передается методу в качестве
аргумента. Метод возвращает добавленного пользователя, полученного из Json-а.
    По id вернувшегося пользователя проверяем успешность выполнения запроса: id пользователя = 11, на единицу
больше, чем самый большой id на сайте (10).

    Метод updateUser(int id, User user) обновляет данные о пользователе с идентификатором
id (параметр метода). "Обновленный" пользователь передается методу в качестве аргумента. Метод возвращает
обновленного пользователя, полученного из Json-а, который идентичный переданному в теле запроса. Идентификатор
вернувшегося пользователя соответствует идентификатору пользователя, которого "обновили".

    Метод deleteUserById (int id) удаляет пользователя по заданному идентификатору id. Метод 
возвращает статус выполнения запроса.

    Метод getUsers() возвращает информацию о всех пользователях в виде списка пользователей.
   
    Метод getUserById(int id) возвращает информацию об определенном пользователе по идентификатору 
пользователя id, который передается методу в качестве аргумента.

    Метод getUserByUsername(String username) возвращает информацию об определенном пользователе по
имени пользователя username, которое передается методу в качестве аргумента.
   
    Задание 2.

    Метод getComments(int x) принимает в качестве аргумента идентификатор определенного пользователя x.
Последний пост этого пользователя имеет номер y=x*10. Метод возвращает все комментарии к посту в виде списка
комментариев под номером y, а также записывает их в файл под именем "user-x-post-y-comments.json". Несколько полученных
в результате выполнения программы файлов находятся в папке resources: user-1-post-10-comments.json, 
user-3-post-30-comments.json, user-7-post-70-comments.json.

    Задание 3.

    Метод getTodosForUser(int id) выводит все открытые задачи для пользователя с идентификатором id в виде списка задач.
Условие выводить только задачи, у которых completed = false, указывается как query-параметр "?completed=false".
